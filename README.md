azion-python-sdk
=========================================================

A Python SDK to provides a pure interface for the [Azion REST API v2](https://www.azion.com.br/developers/api-v2/)

azion-python-sdk is developed using Python 2.7 only.

Installation
------------

    $ pip install azion-sdk
